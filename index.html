Интернет-магазин

Интернет-магазин интегрирован в Telegram MiniApp: фронт реализован на CSS, JS и Bootstrap. Бэкенд — Django с встроенной админкой для управления товарами и заказами.

Исользовались: Django-redis для кэширования, Django signals для обработки событий(удаление корзины, элемента корзины, заказа, переход товара из корзины в заказы и др.).

Сервис обратной связи

Исользовались: Django-redis для кэширования, Celery для асинхронной обработки задач (отправка нового отзыва в чат-бота админам), Django signals для обработки событий.
